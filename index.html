  • How to use it??

    • There are three modes you can use:

    • "Hard mode" and "Easy mode". They both blocking popups that interfere with your reading experience. Easy mode leaves navigation bar on page, so u can change topic. Hard mode does not. There are plenty of websites that deserve it.

    • These modes are available for all pages on current website or for every website you visit while given mode is active. They are "this site" or "everywhere". "Everywhere" mode temporarily removed. Will back in version 2.0.0.

    • Last mode is "I JUST WANT TO READ!" - If nothing else helps. It disables all things that can interrupt you from reading. Works for current page until u reload it or leave in any way.

  • Cons and pros

    • Ultra-lightweight. Ultra-simple. Open source. Does things another tools do not.

    • Part of sites use same mechanics as popups do. They can work not properly. However, nothing serious can happen. Turn extension off for this website, reload page, send me website. We will fix it up!

    • PopUpOFF does not block content ad. Use it in pair with any adblocker.
